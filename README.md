# Chess Board

Bu projede, satranç tahtasında at (knight) taşının sadece hareket edebileceği alanları gösterecek şekilde bir algoritma geliştirilmiştir. Satrançta at L şeklinde (2 kare bir yöne, ardından 1 kare dik yöne) hareket eder. Projede bu kurala göre, kullanıcı bir kare seçtiğinde sadece atın gidebileceği alanlar yeşil renkle vurgulanır. At, kurallar dışındaki herhangi bir kareye hareket edemez.

In this project, an algorithm is developed to display only the valid moves of the knight on a chessboard. In chess, the knight moves in an L-shape (two squares in one direction and then one square perpendicular). According to this rule, when a user selects a square, the valid target squares for the knight are highlighted in green. The knight cannot move to any square that does not comply with the standard rules.
